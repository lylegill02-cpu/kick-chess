# KICK CHESS Hybrid Competition Rules v1.0

**Effective:** June 2026  
**Format:** Alternating kickboxing rounds and chess games with linked consequences

---

## 1. Overview

KICK CHESS applies the hybrid model to **kickboxing**: ring performance modifies chess conditions. Competitors alternate:

1. **Ring Phase** — timed sparring or technical rounds (no gi)
2. **Board Phase** — standard chess (rapid or blitz)

---

## 2. The Ring → Board Spectrum

| Range | State | Board Effect |
|-------|-------|--------------|
| 0–29 | Ring-heavy | −30 sec, no takebacks |
| 30–44 | Fatigued | −15 sec |
| 45–55 | Neutral | Standard time |
| 56–69 | Composed | +5 sec |
| 70–100 | Board-ready | +10 sec |

---

## 3. Ring Phase Scoring (RQS)

| Outcome | RQS | Spectrum Δ |
|---------|-----|------------|
| Clean counters / ring control | +15 to +20 | Toward Board |
| Solid defense, good exits | +5 to +10 | Slight toward Board |
| Neutral exchange | 0 | No change |
| Poor defense, eating shots | −5 to −10 | Toward Ring |
| Knockdown, stalling, foul | −15 to −25 | Strong toward Ring |

---

## 4. Graduated Penalties (Recovery Tiers)

Same tier structure as Roll Chess — Light / Medium / Heavy — applied to ring mistakes before each board game.

---

## 5. Match Structure

**Standard:** 3-min ring → 10+5 board → 3-min ring → 10+5 board  
**Sprint:** 2-min ring → 5+3 blitz

---

## 6. Trainer Alignment

The [Kick Chess Trainer v0.1](https://lylegill02-cpu.github.io/kick-chess/trainer.html) simulates ring → board with Recovery Mode tiers.
